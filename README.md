# C360N - عبدالعزيز العاصي القابضة

## 🚗 حول المشروع

موقع ويب احترافي لمنتج **C360N** - نظام غسيل السيارات الذكي بتقنية الذراع الدوار 360 درجة.

يقدم الموقع:
- عرض شامل لمواصفات المنتج
- لوحة كفاءة الأداء والموارد
- أدوات ذكية مدعومة بـ Google Gemini AI
- معلومات الضمان والدعم الفني
- طرق التواصل والمبيعات

## 🎨 الميزات

### 🌐 تصميم استجابي
- تصميم كامل يدعم جميع الأجهزة (الهاتف، اللوحة، سطح المكتب)
- واجهة عربية (RTL) احترافية
- تجربة مستخدم سلسة وسريعة

### 🤖 أدوات ذكية (AI Suite)
يحتوي الموقع على 3 أدوات مدعومة بـ Google Gemini:

1. **مخطط الاستثمار الذكي** 💰
   - دراسة جدوى فورية لأي موقع
   - حساب الأرباح المتوقعة بناءً على ساعات العمل
   - نصائح لتحسين الربحية

2. **مساعد الصيانة الفني** 🔧
   - استشارات فنية متقدمة
   - إرشادات الصيانة الوقائية
   - حلول لأكواد الأخطاء

3. **مولد منشورات التسويق** 📱
   - إنشاء محتوى احترافي
   - عروض ترويجية مخصصة
   - محتوى تعليمي عن حماية الطلاء

### 📊 لوحة الأداء
- رسوم بيانية تفاعلية (Chart.js)
- مقارنة استهلاك المياه والكهرباء
- معايير الإنتاجية الرئيسية

## 🛠️ التكنولوجيا المستخدمة

- **HTML5** - البنية الأساسية
- **Tailwind CSS** - التصميم والاستجابة
- **Chart.js** - الرسوم البيانية التفاعلية
- **Google Gemini API** - الذكاء الاصطناعي
- **Font Tajawal** - خطوط عربية احترافية

## 🚀 البدء السريع

### المتطلبات
- مفتاح API من Google Gemini (اختياري - بدون مفتاح تعمل معظم الميزات)
- متصفح ويب حديث

### التثبيت

1. استنسخ المستودع:
```bash
git clone https://github.com/abdullaziz-alrashdan/aacom.git
```

2. افتح `index.html` في متصفحك مباشرة أو استخدم خادم محلي:
```bash
python -m http.server 8000
# أو مع Python 3
python3 -m http.server 8000
```

3. **لتفعيل أدوات AI الكاملة:**
   - احصل على مفتاح API من [Google AI Studio](https://aistudio.google.com/app/apikey)
   - أضف المفتاح في السطر التالي من `index.html`:
   ```javascript
   const apiKey = "YOUR_GEMINI_API_KEY_HERE";
   ```
